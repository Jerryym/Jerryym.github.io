Nest Framework v0.1.0开发笔记

2024-02-24
Nest Framework
6.9k words
  • C++
  • Qt

Nest Framework v0.1.0计划使用分层架构设计，实现框架核心与UI分离。

  • Nest：此项目为框架核心，是一个 .lib项目。内部包含Application(应用程序类)、基于spdlog库封装的Log日志模块。
  • NestUI：此项目为框架UI库，是一个 .dll项目。内部包含MainWindow(纯虚类)、DockWidget(纯虚类)、DockWidgetManager(DockWidget窗口管理类)。

项目仓库：NestFramework

Nest

Nest项目为框架核心，是一个 .lib项目。内部包含Application(应用程序类)、基于spdlog库封装的Log日志模块。

应用程序类(Application)

  • 源代码

    • Application.h

    • Application.cpp

  • 此类继承于Qt的 QApplication类，用于管理应用程序和主窗口。

  • 使用宏 Q_OBJECT，使得Qt的元对象系统支持 Application类，能够使用信号和槽机制。

  • 使用单例模式，定义了一个应用程序单例 static Application* s_Instance，确保程序运行时，当且只有一个应用程序存在。

  • std::unique_ptr<QMainWindow> m_Window：通过智能指针管理应用程序的主窗口，且通过 std::unique_ptr 独占所管理对象的所有权，保证应用程序运行时只有一个主窗口。

  • bool m_bRunning：用于控制应用程序的运行状态。

日志模块(Log)

  • 源代码

    • Log.h

    • Log.cpp

  • Log类是基于spdlog库封装的Log日志模块。

  • spdlog使用说明文档：spdlog wiki

NestUI

NestUI项目为框架UI库，是一个 .dll项目。内部包含MainWindow(纯虚类)、DockWidget(纯虚类)、DockWidgetManager(DockWidget窗口管理类)。

主窗口(MainWindow)

  • 源代码

  • 此类继承于Qt的 QMainWindow类，用于设计和管理主窗口。

  • 设计 sWindowProps结构体，用于设置主窗口窗口属性。

停靠窗口(DockWidget)

  • 源代码

  • 此类继承于Qt的 QDockWidget类，用于设计停靠窗口。

  • 设计 clearWidget()和 refreshWidget()，用于清空或刷新继承于 DockWidget的停靠窗口中的内容。

DockWidget窗口管理(DockWidgetManager)

  • 源代码

    • DockWidgetManager.h

    • DockWidgetManager.cpp

  • 此类用于管理主窗口中的停靠窗口对象。

  • 使用单例模式，定义了一个DockWidget窗口管理单例 static DockWidgetManager* s_Instance，确保程序运行时，每个主窗口当且只有一个DockManager存在。

  • QMap<QString, QDockWidget*> m_DockWidgetMap：用于存储创建的dockWidget对象，对通过dockWidget对应的窗口标题与其对象对应。
